TABLAS
Se deben hacer 3 tablas
- Articulos id, nombre estatus
    Solo se CREAN desde la base de datos
- Poliza / se genera cuando hay productos dañados o extraviados
    Se CREAN, EDITAN y ELIMINA
    si ELIMIAR la cantidad se suma al STOCK
    si se EDITA cambia cantidad y estado
    si se CREA cambia la cantidad del STOCK
- Moviminetos (Registro de Altas y Bajas), se registran todos
    Crea uno cada que se CREA, EDITA o ELIMINA algo de POLIZA
        sea a registrado con estatus de Dañado/Perdido y cantidad.
        se elimina cuando se haya encontrado el articulo.

NOTA
La poliza crea para cambiar la cantidad del STOCK del Articulos,
validacion: si tengo 2 articulos no puedo registrar 3 dañados o perdidos
